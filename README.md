## 🚀 Deploy

### Vercel (recommended — auto-detects static)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Feder-prog%2Fstatesite-preview)

Or manually:
1. [vercel.com/new](https://vercel.com/new) → Import Git Repository
2. Select `eder-prog/statesite-preview`
3. Framework Preset: **Other** (Vercel auto-detects static — no build needed)
4. Deploy

The `vercel.json` config enables:
- Clean URLs (`/privacy` instead of `/privacy.html`)
- 1-year cache on images (jpg/png/svg/webp/gif)
- Security headers (X-Content-Type-Options, X-Frame-Options, Referrer-Policy)

## 🎨 Tech & features

- **Vanilla HTML + CSS + JS** — zero JS framework, no build step
- **Google Fonts** Inter (only external dep)
- **Responsive mobile-first** — breakpoints at 480/768/1024/1200/1280
- **Adaptive header** — transparent glass on hero, white opaque on scroll, logo morphs wordmark → icon
- **A2P 10DLC compliant form** — SMS consent (STOP/HELP/rates/Privacy/Terms links), inline + Final CTA duplicate
- **Project gallery** — 11 before/after with filter tabs + lightbox popup (←/→/ESC nav)
- **Hover effects** — service cards reveal, gallery before↔after swap, testimonials CSS slider auto-scroll
- **Touch fallbacks** via `@media (hover: none)` for all hover effects
- **Reduced-motion** respect for all animations
- **Scroll-reveal + count-up** for stats (20+, 100%)
